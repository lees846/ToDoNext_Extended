A tool for you!

Sometimes when we have a lot on our plate it can be so overwhelming that we don't even know where to start. I refer to this as "thrashing".

"Thrashing" is a term used in computer science to describe when a system is given so many tasks to manage simultaneously that it overwhelms the memory trying to decide what to tackle first. This often causes reduced functionality or system failure.

Fundamentally, I feel that humans do the same thing (I know I do all the time). Sometimes we just need a tool to help us weigh our options, and make an informed suggestion for where to begin. Because of this, I decided to create a small system that may be able to help users like you with this process.

Please note that the output from this site does just that: provides suggestions. Each time something generated behind the scenes is presented, it is merely something for you, the viewer of the screen and the completer of tasks, to respond to. If you receive a suggestion and think it's a terrible idea, then congratulations! You have just eliminated something which you do not want to start with :)
